Every drain in production should be wrapped in the **drain pipeline** — `createDrainPipeline()`. It batches events, retries on transient failures, and drops the oldest events when the buffer overflows. Without it, you make one HTTP request per emitted event, which doesn't scale beyond local dev.

Wrap my evlog drain in the drain pipeline so it batches, retries, and survives transient failures.

- Import `createDrainPipeline` from `evlog/pipeline`
- Create a pipeline with batch size and interval (`{ size: 50, intervalMs: 2000 }` are sane defaults) and retry options (`{ maxAttempts: 3 }` with exponential backoff)
- Wrap each underlying drain by calling `pipeline(myDrain)` — the result implements the same Drain contract
- For multiple destinations, pass them in one pipeline as a fanout `pipeline([drainA, drainB])`; cheaper than two pipelines
- On shutdown, call `pipeline.flush()` and `pipeline.dispose()` to drain in-flight events (frameworks with proper lifecycle do this automatically)
- Always use the pipeline in production — direct drains make one HTTP call per event and fall over fast

Docs: https://www.evlog.dev/build-on-top/drain-pipeline

## Quick example

```ts
import { createDrainPipeline } from 'evlog/pipeline'
import { createAxiomDrain } from 'evlog/axiom'

const pipeline = createDrainPipeline<DrainContext>({
  batch: { size: 50, intervalMs: 2000 },
  retry: { maxAttempts: 3 },
})

const drain = pipeline(createAxiomDrain())

// Use `drain` wherever you'd register a drain (Nitro hook, initLogger, etc.)
```

## Fanout pattern

```ts
const drain = pipeline(
  createAxiomDrain(),
  createDatadogDrain(),
  createSentryDrain(),
  createFsDrain(),
)
```

All four destinations receive every event in the same batch. See [Fanout & multi-drain](/build-on-top/fanout-and-multi-drain) for the full recipe.

## Common pitfalls

- **Don't forget `drain.flush()` on shutdown** — buffered events are lost otherwise
- **Tune `batch.size` to match your provider's recommended payload** — too small wastes overhead, too big risks rejection
